# Zyfront Desktop 使用指南

## 概述

Zyfront Desktop 是一个强大的多智能体协作平台，提供可视化编排、自动化任务分配和多种协作/对抗模式。

## 功能特性

### 1. 五种协作/对抗模式

#### 辩论模式
- 正反双方进行观点交锋
- 评委进行评分和裁决
- 支持观点分析和总结+

#### 红蓝攻防模式
- 红队发起攻击策略
- 蓝队进行防御应对
- 漏洞记录和风险评估

#### 竞赛冲刺模式
- 多智能体并行执行任务
- 实时评分和排名
- 冠军裁决机制

#### 回合博弈模式
- 策略继承和进化
- 决策树记录
- 收敛判断机制

#### 混合评审模式
- 多方案提交和讨论
- 投票和评分机制
- 分歧记录和最终裁定

### 2. 可视化编排画布

- **节点拖拽**：轻松拖拽 Agent 和任务节点
- **连线功能**：建立任务依赖关系
- **属性编辑**：配置节点属性和参数
- **画布操作**：缩放、平移、对齐等

### 3. 自动编排能力

- **目标解析**：智能分析用户目标
- **自动组队**：根据目标自动生成团队结构
- **任务拆分**：将大目标分解为可执行任务
- **策略选择**：自动选择合适的调度和收敛策略

### 4. 快照与恢复

- **手动快照**：随时保存当前状态
- **自动快照**：关键操作后自动保存
- **快照恢复**：从历史快照恢复状态
- **版本管理**：管理多个版本的快照

### 5. 错误恢复机制

- **错误捕获**：自动捕获和分类错误
- **错误翻译**：将技术错误转换为用户友好提示
- **恢复建议**：提供具体的恢复步骤
- **自动回退**：从最近的快照恢复

## 快速开始

### 1. 创建智能体

1. 点击"新建Agent"按钮
2. 填写 Agent 信息：
   - 名称：Agent 的名称
   - 角色：选择合适的角色类型（架构师、分析师、开发者等）
   - 描述：Agent 的功能描述
   - 技能：选择 Agent 具备的技能
   - 协作模式：选择默认的协作模式
3. 点击"创建智能体"

### 2. 构建团队

1. 点击"新建团队"按钮
2. 填写团队信息：
   - 团队名称
   - 描述
3. 选择团队成员
4. 点击"创建团队"

### 3. 开始协作

#### 方法一：手动编排
1. 切换到"可视化编排"标签
2. 点击"添加任务"按钮创建任务
3. 拖拽任务和 Agent 节点
4. 建立任务依赖关系
5. 点击启动按钮执行

#### 方法二：自动编排
1. 在"可视化编排"标签中
2. 点击"自动编排"按钮
3. 系统会自动：
   - 分析目标
   - 生成团队
   - 拆分任务
   - 建立依赖关系

## 界面指南

### 顶部导航栏

- **智能体看板**：查看 Agent 和团队状态（竞技场视图）
- **可视化编排**：手动编排任务和 Agent
- **智能体管理**：管理 Agent 和资源

### 左侧面板

- **时间线**：查看操作历史
- **协作摘要**：实时统计信息

### 右侧面板

- **模式选择器**：切换协作/对抗模式
- **Agent 状态**：Agent 实时负载
- **快照列表**：查看和恢复历史快照
- **错误恢复**：处理错误和异常

### 底部工具栏

- **控制按钮**：播放、暂停、重置
- **速度控制**：1x、2x、4x、8x
- **快捷键**：
  - `P`：暂停/播放
  - `R`：重置
  - `M`：切换模式
  - `Tab`：切换视图

## 常见问题

### Q: 如何切换不同的协作模式？
A: 在右侧面板的模式选择器中点击需要的模式即可。

### Q: 自动编排失败怎么办？
A: 可以：
1. 检查错误恢复面板获取详细信息
2. 使用快照恢复功能回滚
3. 尝试手动编排

### Q: Agent 显示异常状态怎么办？
A: 
1. 尝试使用"重连会话"按钮
2. 检查 Agent 资源使用情况
3. 使用快照恢复到正常状态

### Q: 如何导出编排方案？
A: 当前可以通过保存快照来保存编排方案，导出功能正在开发中。
