一、动画与特效概述

1.1 设计原则

复古像素风格: 保持8-bit/16-bit复古游戏的视觉感受

低帧率动画: 使用12fps逐帧动画保持复古感

即时反馈: 所有交互操作都有即时的视觉反馈

性能优先: 确保动画流畅，目标60fps

1.2 动画分类

类别 描述 技术实现

角色动画 Agent角色的各种状态动画 CSS动画/Canvas精灵图

特效动画 爆炸、光束、粒子等特效 Canvas粒子系统

UI动画 界面元素的过渡动画 CSS过渡/动画

环境动画 背景、扫描线等环境效果 CSS/Canvas
